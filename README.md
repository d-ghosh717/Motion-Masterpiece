## 📖 Overview

**Motion Masterpiece** is a React-based animation showcase and developer toolkit featuring interactive UI patterns, 3D effects, and production-ready code snippets.

### Purpose

* Demonstrate modern animation techniques
* Provide reusable code snippets
* Combine design and engineering
* Offer a real-time playground

---

## ✨ Features

### 🚀 Core Experience

* Custom cursor (magnetic + smooth tracking)
* Three.js particle background
* Smooth scrolling (Lenis)
* Glassmorphism UI
* Performance toggle mode

---

### 🎨 Interactive Sections

| Section       | Description        | Tech             |
| ------------- | ------------------ | ---------------- |
| Hero          | Animated landing   | Three.js, Motion |
| Glass Gallery | Morphing cards     | CSS, Motion      |
| Hover Lab     | Button experiments | Motion           |
| Scroll Story  | Scroll animations  | GSAP             |
| Grid Morph    | FLIP grid          | Motion           |
| Depth Zone    | 3D cards           | CSS 3D           |
| Forms         | Animated UX        | React Hook Form  |
| Theme Lab     | Dynamic themes     | Zustand          |

## 📦 Installation

### Requirements

* Node 18+
* npm / yarn / pnpm

### Setup

```bash
git clone https://github.com/yourusername/motion-masterpiece.git
cd motion-masterpiece
npm install
npm run dev
```

### Build

```bash
npm run build
npm run preview
```
